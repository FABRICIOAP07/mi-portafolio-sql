#  Notas promedio

 Aplicación en Python que permite ingresar notas numéricas y calcular el promedio.

## Funcionalidades
- Permite ingresar múltiples notas.
- Finaliza la entrada con la tecla `x`.
- Muestra la lista de notas ingresadas y el promedio.
- Manejo básico de validaciones (evita entradas no numéricas).
