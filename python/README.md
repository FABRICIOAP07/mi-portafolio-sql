#Proyectos de Python

## Mini Proyectos

### **07_calculadora/**
Calculadora con funciones para sumar, restar, multiplicar y dividir.

### **08_crud_tareas/**
CRUD completo en consola para gestionar tareas (crear, leer, actualizar, eliminar).

### **09_conversor_demonedas/**
Conversor de USD ↔ PEN con tipo de cambio configurable.

### **10_adivinadorde_numero/**
Juego interactivo donde el usuario intenta adivinar un número aleatorio.

### **11_analizador_texto/**
Analiza un texto e identifica número de palabras, caracteres y otras métricas.
