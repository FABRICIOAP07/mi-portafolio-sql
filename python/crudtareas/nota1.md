# CRUD de Tareas en Consola

Este proyecto es una gestión de tareas que funciona desde la consola.  
Lo hice para practicar los conceptos como:

- Listas
- Funciones
- Bucles
- Manejo de excepciones
- Menús interactivos
- CRUD (Crear, Leer, Actualizar, Eliminar)



## Funciones principales

- **Agregar tareas**
- **Mostrar todas las tareas**
- **Editar una tarea existente**
- **Eliminar una tarea**
- **Menú interactivo**
