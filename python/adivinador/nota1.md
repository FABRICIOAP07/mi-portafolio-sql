# Adivina el Número

Juego donde el usuario debe adivinar un número generado aleatoriamente por el programa.

Este proyecto permite practicar:

- Uso del módulo `random`
- Bucles `while`
- Validaciones de entrada
- Lógica condicional
- Contadores

---

## ¿Cómo jugar?

El programa genera un número entre **1 y 100**.  
Debes ir probando números hasta acertar.  
El sistema te dirá si estás:

- "Muy bajo"
- "Muy alto"
- "Correcto"

También mostrará cuánto tardaste en adivinarlo.
