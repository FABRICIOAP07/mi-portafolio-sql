# Calculadora en Python
Este proyecto es una calculadora hecha en Python:
- Funciones
- Entradas por teclado
- Condicionales
- Organización de un proyecto simple

## Funcionalidades

- Suma
- Resta
- Multiplicación
- División con validación para evitar división por cero
