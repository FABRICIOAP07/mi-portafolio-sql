
# Conversor de Moneda (USD ↔ PEN)

Este proyecto permite convertir dólares a soles y soles a dólares desde una interfaz de consola.

Fue creado para practicar:

- Funciones en Python
- Manejo de errores
- Entrada de datos por consola
- Lógica condicional
- Formateo de números

---

## Funcionalidades

- Convertir **USD -> PEN**
- Convertir **PEN -> USD**
- Validación de entrada
- Tasa de cambio editable en el código
