# Generador de Contraseñas
Mini herramienta en Python para generar contraseñas seguras de forma aleatoria.
##  Funcionalidades
- Genera contraseñas con letras en mayúsculas o minúsculas, números y símbolos.
- Permite elegir la longitud deseada.
- Uso sencillo desde consola.
